## .ca+信息

投稿一条回声洞。

```消息:no-line-numbers
.ca你好，世界！
这是第一条回声洞。
```

```消息:no-line-numbers
回声洞 #1 添加成功~
```

:::warning
如果尝试添加空信息，会得到回复：

>不能添加空信息！
:::

## .cq

随机查询一个回声洞。

```消息:no-line-numbers
回声洞 #77

早安，这个世界

--洛天依(86*******)
at 24/01/16 15:03:29
```

## .ci+编号

查询指定 ID 的回声洞。

```消息:no-line-numbers
.ci 1
```

```消息:no-line-numbers
回声洞 #1

你好，世界！
这是第一条回声洞。

--洛雨辰~✨(14********)
from Unknown(-1)
已被捡起 0 次
at 24/01/16 11:19:12
```

:::warning
- 如果没有有效数字，会得到回复：

  >解析失败……参数是不是没有填数字或者是填的不是数字？

- 如果查询超出范围，会得到回复：

  >你所查询的回声洞不在范围里呢，现在共有xxx条回声洞~
:::
